# SALES-INSIGHTS
A computer hardware business is facing challenges in dynamically changing market. Sales director decides to invest in data analysis project and he would like to build power BI dashboard that can give him real time sales insights. "
 
  solution: "This project cleans raw sales data and delivers real-time insights through interactive dashboards."

 The impact: "Helps managers make data-driven decisions on inventory, pricing, and promotions."

# OBJECTIVES:
Provide real-time visibility into sales performance.

Identify top-selling products and top customers by revenue.

Track regional trends and customer behavior by years

Support agile decision-making in a dynamic market.

# FEATURES
Interactive sales dashboard (Power BI )

Top 5 products and top 5 customers in relation to revenue.

Trend analysis and performance metrics by years.
